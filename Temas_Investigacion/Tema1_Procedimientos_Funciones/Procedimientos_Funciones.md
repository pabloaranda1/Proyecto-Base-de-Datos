1. Introducción

Dentro de SQL Server, los procedimientos almacenados y las funciones son herramientas fundamentales para encapsular lógica, validar datos y mejorar la eficiencia de las operaciones.
Este trabajo presenta el desarrollo de procedimientos y funciones relacionados al manejo de Publicaciones dentro del proyecto UNIVIA, una plataforma académica para compartir materiales.

El objetivo principal es demostrar el correcto uso de estos objetos de base de datos, evidenciando:

    .Organización y modularización de la lógica SQL
    .Validación de integridad antes de modificar datos
    .Recuperación de métricas mediante funciones reutilizables
    .Implementación correcta de operaciones CRUD guiadas por buenas prácticas
